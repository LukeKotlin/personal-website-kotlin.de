Workout App

The workout App was one of my first small side projects to improve my android development skills. Each workout lasts 30 seconds follows by a 10 second break. During the break the next exercise is announced by the media player of your phone. To exit the workout use the backbutton and a custom dialog will open. If you finish the exercise the finishing time will stored in a SQLite database, which you can find under history. Moreover you can use the BMI calculator to find out if you overweighted or underweighted.

github: https://github.com/LukeKotlin/workout-app

tech stack: android studio, kotlin, gradle, SQLite

Kotlin Tinder clone

A side project where I converted an excisting java application into kotlin.

Kotlin Tinder clone

This application was one of my first side projects where I convertet an excisting java tinder clone into kotlin. The app contains a login and a registration option. If you´re using the app the first time you have to registrate yourself. After that your login data is stored in a database on firebase. Depending on which gender your choose in the registration the opposite gender is shown on the start screen. I used the SwipeCards libary to display the original swipe feature of Tinder. A short toast message will show you if you matched somebody else. Beside from just swiping you can also chat with your matches or edit your profile details for example your profile picture.

github: https://github.com/LukeKotlin/tinder-clone-kotlin

tech stack: android studio, java, kotlin, gradle, firebase
